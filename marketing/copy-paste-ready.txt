--- NOTES FOR REVIEWER ---

NearMe is a location-based event discovery app. No account or login is required — the app uses the device's location (with explicit user permission) to display events happening nearby. All events are sourced from public APIs (Ticketmaster Discovery API, SeatGeek Events API, Google Places) and publicly available venue websites. No user-generated content.

The optional subscription (NearMe Premium) is implemented via Apple StoreKit with a 7-day free trial into an annual plan ($49.99/year), or a weekly plan ($4.99/week) with a 3-day free trial. All core features are functional during the trial period. Trial users get full access.

To test:
1. Allow location access on launch
2. Complete the 7-step onboarding flow (goals, vibe, social style, schedule, blocker, budget)
3. Wait through the brief "AI is curating your feed" loading screen
4. View the personalized teaser with one recommended event
5. Start the free trial to enter the main app
6. Browse the event feed, tap events for details, view map, save events
